# オカメインコの一生を残すアプリ(試作品)

オカメインコとの毎日(体重・食事・様子)を記録し、成長アルバムとして残すためのReact製の試作アプリです。もともとはClaudeのアーティファクト機能で作成しました。

## 主な機能

- プロフィール登録(名前・誕生日・お迎え日など)
- 毎日の記録(元気度・食欲・行動・体重(朝昼晩)・ご飯の量・投薬/おやつ・気になること・ひとこと)
- 過去の記録の追加・編集(連続入力モード対応)
- 体重の推移グラフ(30日ウィンドウ・スワイプ/矢印で前後移動)
- ノートのデータをまとめて読み込む機能(テキスト貼り付け)

## ⚠️ 重要: データ保存についての注意

このアプリは `window.storage` という**Claudeのアーティファクト環境専用の保存API**を使っています。

- **Claude.ai上のアーティファクトとして開いた場合**: データは保存されます(Pro/Max/Team/Enterpriseプランで、公開(Publish)したアーティファクトのみ。ブラウザ/デスクトップ版限定、モバイルアプリでは保存できません)
- **GitHub PagesやローカルPCなど、Claude環境の外で動かした場合**: `window.storage` が存在しないため、保存は一切機能しません。ページを閉じる/リロードすると入力内容は消えます

外部でも保存を効かせたい場合は、`src/App.jsx` 内の `window.storage.get/set/delete` の呼び出しをブラウザの `localStorage` や、Supabase・Firebaseなどの外部データベースに置き換える改修が必要です。

## セットアップ(ローカルで動かす場合)

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開くと確認できます。

## ビルド

```bash
npm run build
```

`dist/` フォルダに静的ファイルが出力されます。GitHub Pagesなどにそのまま配置できます(その場合は `vite.config.js` の `base` をリポジトリ名に合わせて設定してください)。

## 技術スタック

- Recharts(体重グラフ)
